This is a gemini canvas application, bisanya dijalankan di gemini canvas. karena kalau dijalankan diluar gemini canvad bakal butuh apikey.

Ini memiliki banyak fitur keren untuk berinteraksi dengan gemini api, seperti membuat gambar, video, audio, dan lain lain.

## Deployment Details
Aplikasi ini dideploy ke Fly.io:
- **Landing Page**: [https://1affiliate-backend.fly.dev/](https://1affiliate-backend.fly.dev/)
- **Client Dashboard**: [https://1affiliate-backend.fly.dev/client](https://1affiliate-backend.fly.dev/client)
- **Admin Dashboard**: [https://1affiliate-backend.fly.dev/admin](https://1affiliate-backend.fly.dev/admin)
- **API Base**: `https://1affiliate-backend.fly.dev/api`

## Project Structure
- `/server`: Node.js Express backend (Fly.io)
- `/admin`: React Admin Dashboard
- `/client-user`: Vite Single-file user interface
- `/public`: Build assets for server
