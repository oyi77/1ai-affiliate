Gemini AI API Key

Aplikasi ini menggunakan teknologi AI terbaru dari 1affiliate (Gemini). Untuk mengaktifkan fitur chat dan pembuatan gambar, Anda perlu memasukkan API Key Anda sendiri. API Key ini akan disimpan secara aman di browser Anda dan tidak dikirim ke server kami.

Cara Mendapatkan API Key:

  1. Buka Google AI Studio.
  2. Login dengan akun Google Anda.
  3. Klik tombol "Get API Key" di sidebar kiri.
  4. Pilih "Create API key in new project".
  5. Salin kodenya dan masukkan di sini.
